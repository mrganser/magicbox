A web for sharing embedded content.

## Demo

https://themagicbox.onrender.com/

## Getting Started

### Prerequisites

- Minimum engines supported:

```
  "engines": {
    "node": ">=10"
  }
```

### Installing

- Clone the project

- Build command, install dependencies:

  `npm install`

- Start command, start server @ localhost:5000 by default:

  `npm start`
